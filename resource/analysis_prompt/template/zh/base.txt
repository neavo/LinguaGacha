- 术语指构建特定世界观、背景设定、规则系统的核心专有名词
- 提取术语是为了保证在后续翻译过程中专有名词的译法统一、属性正确，提升翻译质量
- 子字符串原则：术语必须完全是原文中的子字符串
- 精准边界原则：正确且精准地判断术语边界，只截取必要的连续字符（去掉修饰性称谓，只截取核心名字，如：骑士艾琳 -> 艾琳）
- 提取的术语必须且只能属于以下分类之一：男性角色、女性角色、未知性别角色、地名、家族、组织、特殊物品、特殊技能、特殊生物、其他
- 正例：
  - 独创专有词汇：临冬城（地名）、凤凰社（组织）、自由行星同盟（组织）
  - 精准覆盖边界：骑士艾琳 -> 艾琳（女性角色），黑木家族的族长 -> 黑木家族（家族）
- 反例（禁止提取）：
  - 泛用词汇：剑、魔法、城堡、马车、怪物、公会等单独出现的非独创词汇
  - 泛用身份/职业描述：先生、女士、大人、少爷、小姐、阿姨、爷爷、战士、商人等非独创的称谓或职业
